# Stock Analysis Agent Module

`stockanalysisagent` is the Spring Boot module that contains the finalized implementation, frontend assets, and runtime configuration.

From the repository root, the usual commands still work:

```bash
./gradlew bootRun
./gradlew test
./gradlew compileJava
```

You can also target the module directly:

```bash
./gradlew :stockanalysisagent:bootRun
./gradlew :stockanalysisagent:test
```
